S02_E05_Thread线程

Main Thread (UI Thread主线程) and Worker Thread

在Worker Thread里绝大多数都不能修改UI Thread，但有例外，如Worker Thread可以修改UI Thread中的ProgressBar

NR: Application not responding

在一个程序当中，UI Thread通常用于接收用户的输入，以及将运算的结果反馈给用户
所以说对于一些可能会产生阻塞的操作，必须放置在Worker Thread当中
